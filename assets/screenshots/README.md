# Screenshots EducMe

Place ici les captures d'écran de ton application EducMe.

## Fichiers attendus :
- `educme-home.png` - Écran d'accueil de l'app
- `educme-quiz.png` - Interface du quiz interactif
- `educme-profile.png` - Page de profil utilisateur
- `educme-fiches.png` - Écran des fiches pédagogiques

## Recommandations :
- Format : PNG ou JPG
- Ratio recommandé : 9:16 (format smartphone)
- Résolution : 1080x1920 ou similaire
- Poids : < 500 KB par image pour des performances optimales

En attendant les vraies captures, le carousel affichera une image placeholder ou sera masqué gracieusement.
